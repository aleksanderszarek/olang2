Introduction

Arrays is the core library of OLang, providing essential functions for operations on arrays, like creating, appending, setting or sorting arrays. By default it is automatically included in every OLang script, so you can use its functions without any additional setup.
You can use the $use arrays command to explicitly include it in your script, but this is not necessary as it is included by default.

Instructions

The arrays library provides a set of instructions that are:

  • declare[](name, type): Creates an array with the specified name and type (int, float, str, bool).
    • declare[](myArray, int)
  • get[](name, index): Returns the value at the specified index in the array. If no index - returns the whole array.
    • get[](myArray)
    get[](myArray, 2)
  • set[](name, index, value): Sets the value at the specified index in the array.
    • set[](myArray, 0, read())
  • append[](name, value): Appends a value to the end of the array.
    • append[](myArray, True)
  • delete[](name, index): Deletes the value at the specified index in the array. If no index - deletes the whole array.
    • delete[](myArray)
    delete[](myArray, 1)
  • length[](name): Returns the length of the array.
    • write("The length of myArray is: ", length[](myArray))
  • sort[](name): Sorts the array in ascending order.
    • sort[](myArray)
  • clear[](name): Clears the array, removing all elements.
    • clear[](myArray)
  • contains[](name, value): Checks if the array contains the specified value and returns a boolean value.
    • if(contains[](myArray, 5)) => write("myArray contains the value 5")
  • reverse[](name): Reverses the order of the elements in the array.
    • reverse[](myArray)
  • indexof[](name, value): Returns the index of the first occurrence of the specified value in the array, or -1 if not found.
    • write("The index of 5 in myArray is: ", indexof[](myArray, 5))
  • lastindexof[](name, value): Returns the index of the last occurrence of the specified value in the array, or -1 if not found.
    • write("The index of 3 in myArray is: ", lastindexof[](myArray, 3))
