Introduction

Convert is the core library of OLang, providing essential functions for converting data. By default it is automatically included in every OLang script, so you can use its functions without any additional setup.
You can use the $use convert command to explicitly include it in your script, but this is not necessary as it is included by default.

Instructions

The convert library provides a set of instructions that are:

  • int(value): Converts the value to an integer.
    • write("23 / 7 is about ", int(23 / 7))
  • float(value): Converts the value to a floating point number (float).
    • set(a, float("3.1415"))
  • str(value): Converts the value to a string.
    • write(str(get(a)))
  • bool(value): Converts the value to an boolean.
    • if(bool(random(0, 1))) => write("true")
  • type(value): Returns the type of the value as a string.
    • write("Type of 'Hello, World' is: ", type("Hello, World"))
