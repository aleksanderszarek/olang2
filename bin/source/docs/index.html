Introduction

OLang is a modern, lightweight programming language designed for simplicity and efficiency. It's ideal for scripting, educational use, and rapid prototyping. OLang features a clean syntax, dynamic typing, and a powerful standard library.

Installation

To install OLang, download the latest binary (bin folder) from the official repository and run the installation script called setup.exe.
Firstly, you will need to accept the license agreement. After that, you will be prompted to choose configuration, default libraries and the installation directory. After installation, you can run OLang scripts by double-clicking the file with .olang extension, or by using 'olang [filename]' command with optional params: --debug for forcing preprocessor debugging and --ignore for ignoring fatal errors and running the code anyways. OLang will automatically open the script in the default command prompt application. You will be able to configure your interpreter there or skip the process by applying default settings.


Configuration

You can configure OLang by editing the config.json file located in the OLang installation directory. This file allows you to set various options such as default libraries to load, error handling, and other preferences.
For example, you can set the default library to load by adding the following line to the config.json file: 

"includeLibraries": ["mainlib", "convert", ...]
You can also set the default error handling method by adding the following line to the config.json file: 
"returnOnFatal": true
This will make OLang return an error code when a fatal error occurs, instead of throwing an exception.
You can also choose to enable or disable the debug mode by adding the following line to the config.json file: 
"preprocessorDebug": true
This will enable the debug mode, which will print debug information to the console when OLang is running.
Lastly, you can choose the directory where OLang will look for libraries by adding the following line to the config.json file: 
"generalLibraryLocation": "path/to/your/libraries"


You can also set all of those options while installing OLang.

Requirements

OLang is designed to be cross-platform and should work on Windows, macOS, and Linux.
Windows 7 or higher is mandatory or macOS 10.12 or higher.

Syntax Overview

OLang is an object-oriented sequential programming language. Loops, variables, functions and instructions are objects, which can be called using exec() instruction from mainlib. OLang requires libraries to run. You can read about all instructions of each included library using buttons below:


You can declare a library using the $use command.
OLang supports functions: 

function greet(name: str){
    write(get(name));
}
exec("greet", read());

OLang supports variables:

declare(a, int, 7);
declare(pi, float);
set(pi, 3.1415);
write(get(pi) - get(a));

OLang supports loops:

for(i:int, 0 => 10, get(i) + 1){
    writeline(get(i), ". Hello, World!");
}

OLang supports conditionals:

declare(i, int, read());
if(get(i) % 2 == 0) => write(get(i), " is divisible by 2!");

Examples

Fibbonacci Sequence

write("Enter the number of terms in the Fibonacci sequence: ");
declare(n, int, read());
declare(a, int, 1);
declare(b, int, 0);
for(i: int, 0 => get(n), get(i) + 1){
    write(get(a), " ");
    set(a, get(a) + get(b));
    set(b, get(a) - get(b));
}


Random Number Generator

$use time;
write("Select start value:\n>> ");
declare(start, int, read());
write("Select end value:\n>> ");
declare(end, int, read());
write(random(get(start), get(end)));


Find substring in string

$use string;
write("Type the main string:\n>> ");
declare(string, str, read());
write("Type the substring you want to find:\n>> ");
declare(substring, str, read());
declare(result, int, find(get(string), get(substring)));
if(get(result) != -1) => write("Substring found starting from index ", get(result), " to index ", int(get(result) + length(get(substring)) - 1));
if(get(result) == -1) => write("Substring not found!");
