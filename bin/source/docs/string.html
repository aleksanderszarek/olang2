Introduction

String is an additional library of OLang, providing functions regarding operations on strings. You can use the $use string command to include it in your script. Without this command, the library will not be included by default.

Instructions

The time library provides a set of instructions that are:

  • concat(str1, str2): Concatenates two strings.
    • declare(strings, str, concat(get(str1), get(str2)))
  • split(str, delimiter): Splits a string into a list of substrings based on the specified delimiter.
    • write(split("Hello, World!", ','))
  • replace(str, old, new): Replaces occurrences of a substring with another substring in the given string.
    • set(text, replace(get(a), " ", "\n"))
  • contains(str, substring): Checks if the string contains the specified substring and returns a boolean value.
    • if(contains(get(a), "!")) => write(get(a))
  • find(str, substring): Finds the substring in the specified string and returns its starting index or -1.
    • write("Substring ", get(a), " found in the string ", get(b), " starting from the index: ", find(get(b), get(a)))
  • length(str): Return the length of the string.
    • if(length("Hello, World!") < 10) => write("Goodbye, World!")
  • startswith(str, substring): Checks if the string starts with the specified substring and returns a boolean value.
    • if(startswith(get(a), "Hello")) => write("Welcome!")
  • endswith(str, substring): Checks if the string ends with the specified substring and returns a boolean value.
    • if(endswith("Hello, World!", "!")) => write("Don't shout!")
