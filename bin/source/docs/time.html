Introduction

Time is an additional library of OLang, providing functions regarding time. You can use the $use time command to include it in your script. Without this command, the library will not be included by default.

Instructions

The time library provides a set of instructions that are:

  • random(start, end): Returns a random value from start to end (both being natural numbers).
    • set(number, random(1, 10))
  • wait n: Waits n seconds before executing the next instruction (n being a natural number).
    • wait 10
    wait get(waitTime) * [random(1, 10) / 10]
  • now(): Returns a current datetime.
    • write(now())
    - dconv(format, date): Converts a date string (YYYY-MM-DD HH:MM:SS) to a specified format (d, m, y, H, M, S).
  • dconv(format, date): Converts a date string (YYYY-MM-DD HH:MM:SS) to a specified format (d, m, y, H, M, S).
    • write(dconv(d, now()))
    write(dconv(H, "2025-12-31 11:41:52"))
