Introduction

Varlib is the core library of OLang, providing essential functions for variable operations. By default it is automatically included in every OLang script, so you can use its functions without any additional setup.
You can use the $use varlib command to explicitly include it in your script, but this is not necessary as it is included by default.

Instructions

The varlib library provides a set of instructions that are:

  • declare(name, type, value): Declares a variable.
    • declare(number, int, int(read()))
    declare(text, str)
    Type accepts: int, float, str, bool; value is optional.
  • get(name): Gets the value of a variable.
    • write("The value of 'number' is: ", get(number))
  • set(name, value): Sets the value of a variable.
    • set(number, get(number) + 1)
  • delete(name): Deletes a variable.
    • delete(text)
  • exists(name): Checks if a variable exists.
    • if(exists(number)) => write("The variable 'number' exists.")
  • typeof(name): Gets the type of a variable.
    • write("The type of 'number' is: ", typeof(number))
