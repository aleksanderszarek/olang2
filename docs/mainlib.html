Introduction

Mainlib is the core library of OLang, providing essential functions for input/output operations, and basic control structures. By default it is automatically included in every OLang script, so you can use its functions without any additional setup.
You can use the $use mainlib command to explicitly include it in your script, but this is not necessary as it is included by default.

Instructions

The mainlib library provides a set of instructions that are:

  • write(value): Outputs the value to the console.
    • write("The value of 'a' / 7 is: ", get(a) / 7, ".")
  • writeline(value): Outputs the value followed by a newline.
    • writeline("Hello, World!")
  • read(): Reads input from the user.
    • declare(name, str, read())
  • exec(functionName, ...args): Executes a function with the given arguments, or a object by an ID.
    • exec("function", get(a), "Argument", true)
    exec(1, get(a), "Argument", true)
  • if(statement) => instruction: If the statement is true, executes the instruction.
    • if(get(a) == 7 or not random(1, 10) % 2) => write("a is equal to 7!")
    Statement accepts symbols: == - equals, > - is greater than, < - is less than, >= - is greater than or equal, <= - is less than or equal, != - is not equal to; and conjunctions: and - both statements must be true for it to be true, or - at least one statement must be true for it to be true, xor - only 1 statement must be true for it to be true, not (before 1 statement) - return the opposite value
  • for(init instruction, iteration condition, iteration instruction){ instructions; }: Declares a for loop
    • for(declare(i, int, 0), get(i) < 10, set(i, get(i) + 1)){ write(get(i)); writeline(""); }
    Init instruction is run before the first iteration starts, iteration condition is checked before each iteration starts - if the statement is false, loop execution stops, and iteration instruction is run for each iteration after all of the instructions were executed.
  • function(arg1: type, arg2: type, ...){ instructions; }: Declare a function object which can be executed
    • function(a: int, b: int){ write(get(a) + get(b)); }
